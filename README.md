# NL2SQL-system


A Natural Language to SQL application that converts user queries in plain English into executable SQL statements using RAG, FAISS, Vector Databases, and LLM-based processing.

## Features
- Natural language query processing
- SQL query generation
- Vector search with FAISS
- Database interaction
- FastAPI backend

## Tech Stack
- Python
- FastAPI
- FAISS
- Vector Database
- BART
- SQL
